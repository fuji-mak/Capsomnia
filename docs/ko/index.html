Caps Lock에
제대로 된 일을 맡기세요

Caps Lock이 물리적인 잠자기 방지 스위치가 됩니다. 켜고 덮개를 닫으면 백그라운드 작업은 그대로 이어집니다.

다운로드 GitHub

Apple silicon · macOS 14 Sonoma 이상 · 서명 및 공증 완료 · 오픈 소스

Caps Lock 켜기

pmset -a disablesleep 1을 실행해 잠자기를 막습니다.

Caps Lock 끄기

pmset -a disablesleep 0을 실행해 평소 잠자기 동작으로 돌아갑니다.

AI 에이전트를 위한 물리 스위치

오래 걸리는 로컬 작업이 있다면 Caps Lock을 켜고 덮개를 닫으세요. 다시 끌 때까지 Capsomnia가 MacBook을 깨워 둡니다. 잠자기 방지 상태는 Caps Lock 표시등으로 바로 확인할 수 있습니다.

  • AI 에이전트

    덮개를 닫아도 Codex나 Claude Code의 긴 작업을 계속 돌립니다.

  • SSH 세션

    원격으로 Mac을 쓰는 도중 잠자기에 들어가 연결이 끊기는 일을 막습니다.

  • 빌드와 다운로드

    오래 걸리는 컴파일과 큰 파일 다운로드가 끝까지 이어집니다.

  • 모바일 연결

    Codex Mobile 같은 모바일 연결을 유지해 작업이 멈추지 않게 합니다.

기능

덮개를 닫아도 Mac은 계속 일합니다

Capsomnia는 덮개를 닫아도 작업이 끊기지 않게 하고 Caps Lock 표시등으로 상태를 알려 주는 작은 Mac 앱입니다. 소스 코드도 모두 공개합니다.

덮개 닫기

덮개를 닫아도 작업은 계속됩니다

Caps Lock을 켜고 MacBook 덮개를 닫아도 로컬 작업은 계속 실행됩니다. 원격 로그인과 네트워크가 켜져 있으면 SSH 접속도 유지됩니다.

물리 표시

Caps Lock 표시등으로 상태 확인

불이 켜져 있으면 잠자기 방지 기능도 켜진 상태입니다. 키보드만 보고 확인할 수 있어 메뉴 막대에 아이콘을 띄우지 않아도 됩니다.

오픈 소스

무료 오픈 소스

MIT 라이선스로 공개합니다. 설치 전에 소스 코드, 패키지 설치 스크립트, helper 명령, 보안 모델을 직접 살펴볼 수 있습니다.

사용자 지정 단축키

원하는 키로 자유롭게

Caps Lock에 이미 다른 기능을 할당했어도 원하는 단축키로 Capsomnia를 켜거나 끌 수 있습니다. Capsomnia가 켜져 있는 동안에는 키보드의 초록색 Caps Lock 표시등이 켜집니다.

다운로드 GitHub

보안 모델

메뉴 막대 앱은 root가 아닌 현재 사용자 권한으로 실행됩니다. 시스템 잠자기 설정을 바꿀 때만 root가 소유한 고정 기능 helper 하나를 비밀번호 없는 sudo로 호출합니다.

  • 네트워크 연결 없음
  • 텔레메트리 없음
  • 계정 불필요

앱이 호출할 수 있는 명령

sudo -n /Library/PrivilegedHelperTools/capsomnia-pmset on
sudo -n /Library/PrivilegedHelperTools/capsomnia-pmset off
sudo -n /Library/PrivilegedHelperTools/capsomnia-pmset display-sleep

패키지로 설치한 앱, helper, 시스템 LaunchAgent의 소유자는 root입니다. sudoers 규칙은 정확히 이 세 명령만 허용합니다.

helper가 실행하는 명령

/usr/bin/pmset -a disablesleep 1
/usr/bin/pmset -a disablesleep 0
/usr/bin/pmset displaysleepnow

on, off, display-sleep 외에는 받아들이지 않습니다.

앱을 종료하면 평소 잠자기 동작으로 돌아갑니다. 잠자기를 막은 채 덮개를 닫으면 발열과 배터리 소모가 늘 수 있으니 통풍, 전원, 실행 시간을 확인하세요.
