让 Caps Lock
真正派上用场

把 Caps Lock 变成实体防休眠开关。开启它，合上屏幕，让后台任务继续运行。

下载 GitHub

Apple silicon · macOS 14 Sonoma+ · Signed & notarized · Open source

Caps Lock 开启

运行 pmset -a disablesleep 1，停用系统睡眠。

Caps Lock 关闭

运行 pmset -a disablesleep 0，恢复正常睡眠。

功能

合上屏幕，Mac 仍可继续工作

Capsomnia 是一款小巧的 Mac 应用，专注于合盖后继续运行任务、通过实体指示灯显示状态，以及透明的开源设计。

合盖运行

合盖后任务继续运行

开启 Caps Lock 并合上 MacBook，本地任务仍会继续运行。在已启用远程登录且网络可用时，你也可以继续通过 SSH 访问 Mac。

实体状态

Caps Lock 指示灯显示状态

指示灯亮起时，防休眠功能已开启。无需显示菜单栏图标，也能直接从键盘确认状态。

安全模型

菜单栏应用以当前用户身份运行，绝不会以 root 身份运行。更改系统睡眠设置需要提升权限，因此 Capsomnia 通过免密码 sudo 调用一个功能固定、由 root 所有的小型辅助程序。

  • 无网络请求
  • 无遥测
  • 无需账户

应用只能调用

sudo -n /Library/PrivilegedHelperTools/capsomnia-pmset on
sudo -n /Library/PrivilegedHelperTools/capsomnia-pmset off
sudo -n /Library/PrivilegedHelperTools/capsomnia-pmset display-sleep

软件包中的应用、辅助程序和系统 LaunchAgent 均由 root 所有。sudoers 规则仅允许执行以下三个完全匹配的命令。

辅助程序只会运行

/usr/bin/pmset -a disablesleep 1
/usr/bin/pmset -a disablesleep 0
/usr/bin/pmset displaysleepnow

仅接受 on、off 和 display-sleep，不会接受其他参数。

退出应用会恢复正常睡眠。停用睡眠并合盖使用可能会增加发热和耗电量，请留意通风、电源和运行时长。
